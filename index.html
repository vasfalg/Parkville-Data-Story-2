Quickly Edit and View Changes on Your Local Computer

HINT: Instead of slowly making edits on GitHub, you can quickly make edits in your local computer and view them instantly in your browser. Here's a list of steps (in HTML format):

  1. Go to your GitHub repo and select Code > Download Zip and uncompress the folder.
  2. Install a free code editor tool. My fav is Atom Editor but there are others, too.
  3. Open the "index.html" file with your code editor and make edits.
  4. In your browser, go to File > Open to view the "index.html" file that is on your local computer.
  5. After making some edits in the "index.html" file, save it, and refresh your browser to view changes. Most edits should appear right away.
  6. After making multiple edits on your local computer, carefully copy and paste the contents of "index.html" in your online GitHub account, and test your published updates online.
